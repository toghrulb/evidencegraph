# EvidenceGraph project instructions

[`AGENTS.md`](AGENTS.md) is the canonical and complete implementation specification for this repository.

Before inspecting, planning, or changing the project, read `AGENTS.md` completely and follow it as though its contents appeared here. When these instructions are consumed by an automated coding tool, this reference is mandatory rather than optional background.

This file deliberately references the canonical source instead of duplicating it, so the two instruction entry points cannot drift apart. A filesystem symlink is avoided because it is not reliably preserved on every Windows checkout.
